Flight Search Interface using Skyscanner's Design System

This project is a flight search interface built using React and Skyscanner's Backpack design system. It demonstrates the implementation of a clean, responsive user interface following Skyscanner's design principles.

Project Structure

The main application is located in the my-app directory, which contains a React application built with Create React App and customized with Skyscanner's components.

Features Implemented

Search interface with from and to city inputs
Flight search functionality with mock results
Responsive design that works on mobile and desktop
Implementation of Skyscanner's Backpack components
Clean and professional user interface
Proper state management using React hooks

Technologies Used

React 18.2.0
Skyscanner Backpack Design System
SCSS for styling
Modern JavaScript
Responsive design principles

Development Details

The project uses Skyscanner's official UI components to create a professional-looking interface
Implemented proper component structure and state management
Added responsive styling for different screen sizes
Used SCSS with Backpack's design tokens and mixins
Followed modern React development practices

Getting Started

Navigate to the my-app directory
Install dependencies with npm install
Start the development server with npm start
The application will run on localhost:3000

The project serves as a demonstration of working with professional-grade UI components and implementing a real-world interface using React and Skyscanner's design system.
